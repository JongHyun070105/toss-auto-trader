# 리스크/안전장치 초안

자동매매는 원금 손실 가능성이 있으므로, 이 프로젝트는 아래 제한을 기본값으로 둔다.

## 기본 제한

- 기본 실행 모드: `DRY_RUN=true`
- 초기 실거래 총자본: 10,000원 이하
- 1회 주문 금액: 1,000~3,000원 이하로 시작
- 일일 최대 주문 횟수: 1~3회
- 일일 손실 한도: 사용자가 정한 금액 도달 시 자동 중지
- 허용 종목: 화이트리스트에 등록한 종목만 주문 가능
- 주문 방식: 처음에는 지정가만 사용

## 금지할 것

- API 키/계좌번호를 코드나 git에 저장
- AI/LLM이 단독으로 매수·매도 결정 후 바로 주문
- 손실 한도 없는 무한 루프 주문
- 장중 급등주/테마주 자동 추격매수
- 빚/신용/미수/레버리지 사용

## 주문 전 체크리스트

실제 주문 전 봇은 최소한 아래를 출력해야 한다.

- 종목
- 매수/매도 방향
- 주문 수량
- 주문 가격
- 예상 주문 금액
- 현재 보유 현금
- 오늘 누적 주문 횟수
- 오늘 누적 손익/손실 한도
- DRY_RUN 여부

## 중지 조건

- API 오류 연속 발생
- 체결 상태 불명확
- 예상보다 큰 주문 금액 계산
- 잔고/현금 조회 실패
- 네트워크 불안정
- 사용자가 수동 중지
